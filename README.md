# Ласкаво просимо до мого проєкту: Система обліку оргтехніки!

## Огляд проєкту

Цей проєкт розроблено в рамках курсу **"Технології прикладного програмування"**. Це повноцінний RESTful веб-застосунок, призначений для управління інвентаризацією оргтехніки (Ієрархія: Місто -> Магазин -> Товар). Він дозволяє користувачам взаємодіяти з базою даних MySQL через захищений веб-інтерфейс.

## Використані технології

* **Мова програмування:** Java 21
* **Фреймворк:** Spring Boot 3 (Web, Security, Data JPA)
* **База даних:** MySQL 8.0
* **Мова розмітки:** HTML (Thymeleaf)
* **Стилізація:** Bootstrap 5
* **Інструмент збірки:** Maven

## Функціонал

* **CRUD операції:** Повне управління магазинами та товарами (Створення, Читання, Оновлення, Видалення) для Адміністраторів.
* **Інтерактивний веб-інтерфейс:** Зручні сторінки з адаптивним дизайном.
* **Рольова безпека:**
    * **Гість (Guest):** Тільки сторінка входу.
    * **Користувач (User):** Доступ до списків лише для перегляду (Read-only).
    * **Адміністратор (Admin):** Повний доступ з можливостями редагування.
* **Безпека даних:** Паролі користувачів зашифровані за допомогою Base64.
* **Автоматична ініціалізація:** Таблиці бази даних та початкові дані створюються автоматично за допомогою скрипта `init.sql`.

## Встановлення та запуск

1.  **Вимоги:** Встановіть Java 17+ та MySQL Server.
2.  **Налаштування бази даних:** Створіть схему з назвою `rgr_orgtechnics` у MySQL Workbench.
3.  **Конфігурація:** Відкрийте `src/main/resources/application.properties` та оновіть `spring.datasource.password` на ваш пароль від MySQL.
4.  **Запуск:** Запустіть файл `RgrApplication.java` в IntelliJ IDEA.
5.  **Доступ:** Відкрийте браузер за адресою `http://localhost:8080/login`.
