# RGR: Orgtechnics Inventory System

## Опис проєкту
RESTful веб-застосунок для обліку оргтехніки (Варіант 5: Місто -> Магазин -> Товар). Розроблено в рамках курсу "Технології прикладного програмування".

## Функціонал
* **Ролі:**
    * `GUEST`: Авторизація.
    * `USER`: Перегляд магазинів та товарів (Read-only).
    * `ADMIN`: Додавання та видалення даних (Full Access).
* **Безпека:**
    * Паролі користувачів закодовані в Base64.
    * Реалізовано захист маршрутів (403 Forbidden для User на сторінках Admin).

## Запуск
1. Створити базу даних за допомогою скрипта `src/main/resources/init.sql`.
2. Налаштувати `application.properties` (вказати свій пароль MySQL).
3. Запустити `RgrApplication.java`.
